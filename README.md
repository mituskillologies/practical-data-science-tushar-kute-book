## üöÄ Getting Started

To run the code examples on your local machine, follow these steps:

### Prerequisites

Ensure you have Python installed along with the following libraries:
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/mituskillologies/practical-data-science-tushar-kute-book.git](https://github.com/mituskillologies/practical-data-science-tushar-kute-book.git)
    ```

2.  **Navigate to the directory:**
    ```bash
    cd practical-data-science-tushar-kute-book
    ```

3.  **Install required libraries:**
    ```bash
    pip install numpy pandas matplotlib seaborn jupyter scikit-learn
    ```

4.  **Launch Jupyter Notebook:**
    ```bash
    jupyter notebook
    ```
